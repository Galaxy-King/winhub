API Key Usage

Create a key, assign module permissions, then send the key with every API request.

1. Create

Use New API Key and save the generated secret immediately. WinHUB shows it only once.

2. Grant

Tick only the module actions and host groups the integration needs. Empty host scope means no host access.

3. Call

Send the key in Authorization Bearer or X-API-Key. Expired keys return 401; missing permissions return 403.

curl -H "Authorization: Bearer wh_your_generated_key" https://your-winhub/api/infrastructure/tasks/all
curl -H "X-API-Key: wh_your_generated_key" https://your-winhub/api/newsletter/config
curl -X POST https://your-winhub/api/infrastructure/templates/TEMPLATE_ID/run \
  -H "Authorization: Bearer wh_your_generated_key" \
  -H "Content-Type: application/json" \
  -d "{\"target_type\":\"host\",\"target_id\":\"HOST_ID\",\"variables\":{\"admin_username\":\"localadmin\"}}"
